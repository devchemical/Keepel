Project Overview

What Keepel is and what problem it solves.

🚗
Vehicle Management
Register and manage multiple vehicles with detailed profiles, specifications, and service histories in one place.
🔧
Maintenance Tracking
Log every service, repair, and inspection with dates, costs, notes, and receipts for complete historical records.
📅
Service Scheduling
Schedule future maintenance tasks and get reminders so you never miss an oil change, tire rotation, or inspection.
📊
Cost Analytics
Monitor maintenance spending over time with charts and reports. Compare costs across vehicles and categories.
📱
Responsive & Fast
Built with Next.js App Router and TailwindCSS. Works seamlessly on desktop, tablet, and mobile devices.
🔐
Secure by Default
Row Level Security, JWT auth with Supabase, rate limiting, and input validation keep your data safe.

Route Protection

Middleware and access rules for application routes.

🔓
Guest Routes /auth/*
Only accessible to unauthenticated users. Authenticated visitors are redirected to / (dashboard).
🔒
Protected Routes /vehicles/*
Require authentication. Unauthenticated users are redirected to /auth/login.

Database Schema

PostgreSQL tables with Row Level Security enabled.

Table Purpose Relationships Security
profiles User profiles (1:1 with auth.users) Linked to auth.users RLS enforced
vehicles User vehicles and specifications 1:N with profiles RLS enforced
maintenance_records Service and repair history 1:N with vehicles RLS enforced

Key Files & Hooks

Quick reference for the most important files and hooks.

Item Path Exposes
useAuth() @/contexts user, profile, isAuthenticated, signOut
useData() @/contexts vehicles, maintenance, CRUD methods
useSupabase() @/contexts Raw Supabase client
useProtectedRoute() @/hooks Redirect to login if unauthenticated
useGuestRoute() @/hooks Redirect to / if authenticated
useDashboardData() @/hooks Dashboard data fetching
useAnalytics() @/hooks @openpanel/nextjs integration
useMediaQuery() @/hooks Responsive breakpoint detection
